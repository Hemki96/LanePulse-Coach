# QA-Berichte

In diesem Verzeichnis werden strukturierte Testberichte für BLE-, UI- und Feldtests abgelegt. Jeder Bericht nutzt das unten stehende Template und benennt sich nach Schema `YYYY-MM-DD_<kurzbeschreibung>.md`.

## Template verwenden
1. Kopie von [`report-template.md`](report-template.md) erstellen.
2. Datei umbenennen (z. B. `2025-11-08_ble-regression.md`).
3. Felder ausfüllen und Pull Request mit Link im QA-Ticket referenzieren.

## Verzeichnisstruktur
```
reports/
├── README.md           # Diese Anleitung
└── report-template.md  # Markdown-Vorlage für neue Berichte
```
