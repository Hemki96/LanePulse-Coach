# Abnahmekriterien (MVP)

- Live-Tracking bis 8 Sensoren stabil
- Anzeige-Latenz ≤ 1 s (Median) in Feldtest
- Auto-Reconnect ≤ 8 s (P95)
- Aktiv/Pause in definierten Szenarien ≥ 90 % korrekt
- Konfigurierbare Kachelkennzahlen, persistente Mappings
- CSV/JSON-Export funktional und korrekt
