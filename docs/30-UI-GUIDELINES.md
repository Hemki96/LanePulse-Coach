# UI-Guidelines

- **Board-Kacheln**: große HR-Zahl, Zonenfarbe als Hintergrund/Border, Trendpfeil
- **Scoreboard**: extrem große Typo, hoher Kontrast, max. 8 Felder
- **Detailpane**: Mini-Timeline (letzte 2 min), Zonenbalken, Recovery-Timer
- **Einstellungen**: Metrik-Checkliste, Zonen-Editor, Schwellen Aktiv/Pause
- **Gesten**: Tap→Detail, Long-Press→Intervallmarke
