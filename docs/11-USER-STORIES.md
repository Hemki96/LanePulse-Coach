# User Stories (Auszug)

- **Als Coach** möchte ich **bis zu 8 Athlet:innen** gleichzeitig sehen, **damit** ich Serien zielgerichtet steuern kann.
- **Als Coach** möchte ich **Dropouts automatisch** abgefangen bekommen, **damit** die Anzeige stabil bleibt.
- **Als Coach** möchte ich **Sensoren dauerhaft zuordnen**, **damit** das Setup vor Ort schnell ist.
- **Als Coach** möchte ich **Kennzahlen konfigurieren**, **damit** ich je Gruppe passende Infos sehe.
- **Als Coach** möchte ich **Aktiv/Pause** automatisch erkannt bekommen, **damit** ich Intervalle objektiv bewerten kann.
- **Als Coach** möchte ich **Daten exportieren**, **damit** ich sie später auswerte oder archiviere.
- **Als Athlet:in** möchte ich meine **HR/Zone groß** sehen, **damit** ich live mein Tempo anpasse.
