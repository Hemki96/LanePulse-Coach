# Use Cases (Kurzform)

## UC-01 Session starten
- **Vorbedingungen:** iPad Bluetooth an, Sensoren geladen.
- **Hauptfluss:** Gruppe wählen → Scan → Mappings laden → fehlende zuweisen → Start.
- **Alternativen:** Keine Sensoren gefunden → Hinweis & Retry.

## UC-02 Live-Monitoring
- Board zeigt HR/Zone; Dropouts → Auto-Reconnect; Coach setzt Intervallmarken.

## UC-03 Export
- Session beenden → Export als CSV/JSON → Ablage lokal.

## UC-04 Scoreboard
- Athleten-Ansicht → AirPlay/HDMI → Großanzeige.
