# Teststrategie

- **Unit-Tests**: Algorithmik (EWMA, Hysterese), Parser, Exporte
- **Integration**: BLE-Mocks/Simulator + echte Polar-Sensoren
- **UI-Tests**: Snapshot/UITest Board & Scoreboard (nach Matrix [testing/test-matrix.md](testing/test-matrix.md))
- **Lasttest**: 8 parallele Streams, Dropouts, Reconnect-Pfade
- **Feldtests**: Siehe 42-FIELD-TEST-PROTOCOL-POOL
- **Dokumentation**: Ergebnisse als Berichte in [testing/reports](testing/reports/README.md)
