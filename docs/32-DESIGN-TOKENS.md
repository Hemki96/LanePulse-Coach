# Design Tokens (Basis)

- **Farben (Zonen):** Z1, Z2, Z3, Z4, Z5 (kontrastreich, farbenblind-freundlich)
- **Typo:** Scoreboard XL/XXL; Board L/XL
- **Abstände:** 8/12/16/24 px Skala
- **Radius:** 12–20 px
- **Icons:** klare Pfeile für Trend, Statuspunkte (grün/gelb/grau)
