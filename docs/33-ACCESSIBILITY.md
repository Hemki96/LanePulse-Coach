# Barrierefreiheit

- **Dynamic Type** unterstützen
- **Kontrast** ≥ WCAG AA
- **Farbenblindheit**: alternative Muster/Borders für Zonen
- **VoiceOver**: klare Labels (Name, HR, Zone, Status)
- **Reduce Motion** respektieren
