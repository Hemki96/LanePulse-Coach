# User Flows (Textskizzen)

1) **Session Start**
   - App öffnen → Gruppe wählen → Scan → Mappings laden → fehlende zuweisen → Start

2) **Live**
   - Board-Überblick → Detailtap → Intervallmarke → Scoreboard umschalten

3) **Export**
   - Session beenden → Export wählen → CSV/JSON speichern

4) **Einstellungen**
   - Metrik-Auswahl, Zonenmodell, Schwellen, Board-Layout
