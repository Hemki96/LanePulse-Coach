# Feldtestprotokoll (Hallenbad)

## Setup
- iPad am Beckenrand (spritzwassergeschützt)
- 2–3 Polar Verity Sense, Ladezustand prüfen
- Board-Layouts wechseln (2×2/3×3)

## Szenen
- Einschwimmen (Z1–Z2), Serien mit Sprints (Z4–Z5), definierte Pausen (30/60/120 s)

## Messungen
- Latenz pro Szene, Dropout-Quote, Reconnect-Zeiten, Erkennungsqualität Aktiv/Pause
