# Security & DSGVO (Offline-Only)

- **Datenhaltung lokal** (On My iPad)
- **Datensparsamkeit**: nur notwendige HR- und Event-Daten
- **Einwilligungen** für Minderjährige (Vorlagen optional)
- **Pseudonymisierung**: Alias statt Klarnamen möglich
- **Rechtebetroffener**: Export/Löschung pro Athlet/Session
- **Löschkonzept**: konfigurierbare Aufbewahrungsdauer
